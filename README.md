# Web 开发示例

Http 常用请求方式：post get put delete

请求+响应   调用+返回结果

- 前端发起请求
- 后端接到请求
  - 过滤前端请求数据
  - 查询数据库
  - 处理数据
  - 返回结果
- 前接收结果
- 前端渲染
